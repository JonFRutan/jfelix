Linux

When I was searching for a Linux distro I had two key things I deemed necessary: Stability and Simplicity.

Personally I like to be able to view my running processes and recognize everything. Slackware was what I was looking for, but a bit intimidating.

So my personal recommendation for a Linux distro is SalixOS which was designed to make Slackware more accessible to newcomers like myself.
