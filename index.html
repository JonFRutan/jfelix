Welcome

This is my website. I purchased my domain from NameCheap and host via nginx on a minimal ubuntu server provided by Oracle Cloud services. (2024-09-10)

Afterwards I used certbot to create a server certificate making the connection HTTPS (2024-09-11)

The site is static HTML and CSS, and will automatically update to match this GitHub repo. This was accomplished with crontab.

The purpose of this site is to be compact and instantly-deployable.
